# Docs - Nexo

## Política de Privacidad

El archivo `privacy-policy.html` debe publicarse en una URL pública para cumplir con los requisitos de Apple App Store.

### Publicar en GitHub Pages

1. Edita `privacy-policy.html` y reemplaza `[TU_EMAIL_DE_CONTACTO]` con tu email.
2. Crea una rama `gh-pages` o usa GitHub Pages desde la carpeta `docs/`:
   - Repo → Settings → Pages → Source: Deploy from a branch
   - Branch: `main` (o `gh-pages`), folder: `/docs`
3. La URL será: `https://TU_USUARIO.github.io/nexo/privacy-policy.html`

### Alternativas

- **Notion:** Crea una página, copia el contenido, hazla pública y usa la URL.
- **Vercel/Netlify:** Sube la carpeta `docs/` como sitio estático.
